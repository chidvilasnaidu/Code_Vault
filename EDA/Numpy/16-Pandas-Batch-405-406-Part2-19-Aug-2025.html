Merge Operations in Pandas¶

INNER JOIN (Intersection)¶

  • Set Theory: $A \cap B$

  • This means elements common to both sets.

  • Venn Diagram: Only the overlapping (middle) part is highlighted.

  • SQL Meaning: Returns rows where there is a match in both the left and right table.

2. FULL JOIN (Union)¶

  • Set Theory: $A \cup B$

  • This means all elements from both sets, with overlap included once.

  • Venn Diagram: Both circles completely highlighted.

  • SQL Meaning: Returns all rows when there is a match in either left or right table. Missing values are filled with NULLs.

3. LEFT JOIN (Left Outer Join)¶

  • Set Theory: $A \cup (A \cap B)$ but keeping everything from A.

  • Simply put: all elements of A plus the intersection.

  • Venn Diagram: Entire left circle is highlighted, including overlap.

  • SQL Meaning: Returns all rows from the left table, and matched rows from the right table (NULLs if no match).

4. RIGHT JOIN (Right Outer Join)¶

  • Set Theory: $B \cup (A \cap B)$ but keeping everything from B.

  • Simply put: all elements of B plus the intersection.

  • Venn Diagram: Entire right circle is highlighted, including overlap.

  • SQL Meaning: Returns all rows from the right table, and matched rows from the left table (NULLs if no match).
